#  SaveMyTime - приложение для создания персонального расписания с алгоритмом распределения задач.
SaveMyTime помогает распределить задачи и цели пользователя, учитывая время на работу, сон, отдых и фиксированные во времени события. 
Уникальный алгоритм подбирает лучшее время для самых важных задач и целей.  

## Функционал приложения:
- Добавление задач и целей, с возможностью настроить их сложность, важность и время, необходимое для их выполнения;
- Добавление фиксированных во времени событий - алгоритм учтет их при распределении задач и целей;
- Учет времени на работу и сон;
- Учет хронотипа и уровня концентрации пользователя, наличие специального теста для их определения;
- Генерация расписания на несколько дней вперед.

### Алгоритм распределяет задачи на основе:
- приоритета и сложности задачи;
- доступных временных окон;
- фиксированных событий;
- уровня концентрации пользователя;
- хронотипа.

## Архитектурные решения: 
- Уникальный алгоритм распределения задач и целей;
- Использование технологий облачной базы данных Firebase;
- Русская и английская локализация;
- Интеграция с телеграм-ботом для отправки отчетов об ошибках;
- Использование MVVM и принципов чистой архитектуры.
